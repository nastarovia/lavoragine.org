# la voragine

haz tu propia versión! necesitas:

(Construido con:)
- Node v8.0.0
- NPM v5.0.0
- Bundler v1.16.0
- Gem v2.7.1

si usas NVM, esto te da las dependencias correctass:
```$bash
nvm use v8.0.0
```

to run:
```$bash
bundle install
npm install
npx bundle exec gulp
```
